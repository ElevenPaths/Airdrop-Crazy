# Aidrop API

## Prerequisites

* To make this server works you need to install first:
        * Python3 (https://www.python.org/download/releases/3.0/)

## Installation

* sudo docker-compose up --build


## Deployment

### In host
* sudo bash deploy.sh
* scp ./"DIRECTORY".zip user@dir:/usr/src

### In server
* ssh user@dir
* cd /usr/src
* sudo bash deploy_host.sh
